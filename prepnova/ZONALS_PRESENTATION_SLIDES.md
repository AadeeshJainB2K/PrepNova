#### **Algorithm 1: AI Mock Test Generation**

```

   - Include topic, difficulty level, exam format
   - Specify question type (MCQ/numerical/subjective)
4. Send request to Google Gemini API
5. Parse AI response to extract:
   - Question text
   - Options (for MCQ)
   - Correct answer
   - Detailed explanation
6. Validate question format and content
7. Store in database with metadata
8. Display to user with timer
9. On submission:
   - Evaluate answer
   - Calculate score
   - Update user statistics
   - Show explanation
```

#### **Algorithm 2: Rank Prediction (ML)**

```
1. Fetch user's performance data:
   - Total questions attempted
   - Accuracy percentage
   - Subject-wise scores
   - Time taken per question
   - Consistency (study streak)
   
2. Calculate weighted features:
   - Accuracy weight: 40%
   - Speed weight: 20%
   - Consistency weight: 20%
   - Topic mastery weight: 20%
   
3. Compare with historical data:
   - Fetch performance of previous year's students
   - Normalize scores on same scale
   
4. Apply linear regression model:
   predictedRank = Î²â‚€ + Î²â‚(accuracy) + Î²â‚‚(speed) + 
                   Î²â‚ƒ(consistency) + Î²â‚„(topicMastery)
   
5. Calculate confidence interval (Â±10%)
6. Generate personalized recommendations
7. Display prediction with trend graph
```

#### **Algorithm 3: AI Chat with Context Memory**

```
1. User sends message
2. Retrieve conversation history from database
3. Build context array:
   conversationHistory = [
     {role: "system", content: "You are an exam tutor..."},
```

#### **3. Rank Prediction Algorithm**

```typescript
// lib/ml/rankPredictor.ts

interface PerformanceData {
  accuracy: number;      // 0-100
  avgSpeed: number;      // seconds per question
  consistency: number;   // study streak days
  topicMastery: number;  // 0-100
}

function predictRank(data: PerformanceData): number {
  // Weights based on correlation analysis
  const weights = {
    accuracy: 0.40,
    speed: 0.20,
    consistency: 0.20,
    topicMastery: 0.20
  };
  
  // Normalize speed (lower is better)
  const normalizedSpeed = 100 - (data.avgSpeed / 3) * 100;
  
  // Calculate composite score
  const score = 
    (data.accuracy * weights.accuracy) +
    (normalizedSpeed * weights.speed) +
    (data.consistency * weights.consistency) +
    (data.topicMastery * weights.topicMastery);
  
  // Convert score to rank (inverse relationship)
  // Score 90+ â†’ Rank < 1000
  // Score 70-90 â†’ Rank 1000-10000
  // Score < 70 â†’ Rank > 10000
  const predictedRank = Math.round(
    100000 / (score + 10)
  );
  
  return predictedRank;
}
```

**How it works:**
- Takes 4 performance metrics as input
- Applies weighted scoring (accuracy most important)
- Normalizes speed (faster = better)
- Calculates composite score
- Converts to predicted rank using inverse formula
- Returns rank estimate with Â±10% confidence
